# **App Name**: AstroVerse

## Core Features:

- Home View: Display a landing page showcasing featured astrophysical events and a brief system overview.
- Event Catalog: Display a searchable and filterable list of astrophysical events from the data.
- Wallet Connection: Enable users to connect their MetaMask wallet to the application.
- Google Sign-In: Allow users to authenticate using Google Sign-In for a seamless experience.
- User Wallet Display: Display user's NFT collection and ASTRO token balance.
- AI Event Suggester: Use an AI tool to suggest the most relevant or interesting real-time event data based on the user's profile and past activity, fetched from a Google-hosted API or NASA.
- NFT Minting: Allow the minting of NFTs representing astrophysical events after ASTRO token approval.

## Style Guidelines:

- Primary color: Deep space blue (#242959) to evoke the vastness of space.
- Background color: Dark, desaturated blue-gray (#292A2D) to provide contrast and focus.
- Accent color: Electric purple (#7B62FF) to highlight interactive elements and calls to action.
- Headline font: 'Space Grotesk' (sans-serif) for a modern, technical feel.
- Body font: 'Inter' (sans-serif) providing clean, readable text for descriptions and details.
- Use clean, outlined icons related to space, astronomy, and blockchain.
- Employ a spacious, card-based layout to highlight each event and NFT. Create a sense of depth by layering graphic elements over the desaturated background.
- Use subtle animations (e.g., fade-in effects, smooth transitions) to enhance the user experience without being distracting.
